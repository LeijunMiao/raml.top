阅读规范

构建您的API

RAML通过将规范转换为包含NodeJS，Java，.NET和Python等几种不同语言的服务器端生成器的代码，可以轻松构建API - 每天都会添加更多内容。您可以在 项目库中查看使用RAML构建API的所有项目，但这里只是几个：

NodeJS

Osprey

Osprey 是一个JavaScript框架，用于快速构建暴露RAML API的应用程序。它基于Node和Express。

更多信息下载


Java

Restlet

借助Restlet Framework强大的路由和过滤功能，统一的客户端和服务器Java API，开发人员可以构建安全且可扩展的RESTful Web API。它适用于所有主要平台（Java SE / EE，Google AppEngine，OSGi，GWT，Android），并提供多种扩展以满足所有开发人员的需求。

更多信息尝试一下


RAML for JAX-RS

RAML for JAX-RS的目标是提供一组工具来使用这些技术来构建API，以便能够基于现有的RAML API定义（代码生成）构建JAVA + JAX-RS应用程序。或其往返，基于现有的JAVA + JAX-RS应用程序（文档）生成RAML API定义。

更多信息下载


.NET

RAML Tools for .NET

RAML Tools for .NET 允许开发人员轻松集成和使用公开RAML定义的API，或者使用先前创建的RAML定义中的契约优先方法创建新的ASP.NET Web API实现。单个工具包为客户端和服务代码生成方案提供支持。这些工具作为Visual Studio扩展提供，允许将RAML简单而自然地集成到正常的开发工作流程中。

更多信息下载


Python

Flask-RAML

Flask-RAML (REST API Markup Language) 生成具有参数转换，响应编码和示例的API服务器。

更多信息下载


raml-python

RAML-python uses NodeJS 为Python生成API框架。

更多信息下载


RAMSES

RAML + Elasticsearch + Pyramid = REST API. Ramses是一个使用RAML生成RESTful API的库。它使用Pyramid和Nefertari提供ElasticSearch支持的视图。

更多信息下载


Mule

APIkit

APIkit是一个声明性工具包，它利用RAML来构建和促进API的实现。它使用RAML作为API的定义，然后可以与基于APIkit Maven和Mule Studio的工具一起使用，用于模拟接口并使用Mule流实现其后端。

更多信息下载


IOT

Nobel

Nobel（代码生成器）基于RAML定义为Arduino板创建REST API。

更多信息下载

RAML是唯一一个旨在以人类可读格式包含完整API生命周期的规范，其中包含代码和设计模式重用。使用RAML，您可以使用一个规范真正 设计, 构建, 测试, 记录, 和 共享 您的API。
