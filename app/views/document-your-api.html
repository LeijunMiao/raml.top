API文档

REST API文档变得简单

RAML使您可以轻松记录REST API，甚至可以更好地保持API文档的同步！有了数百种开源工具，如API控制台，API笔记本和RAML 2，HTML文档可以快速，即时生成，并且几乎每种语言的解析器都可以创建自己的自定义文档和交互式脚本，如 e.Pages 和 Spotify。

API控制台

API控制台提供实时交互式文档，让用户可以实时试用您的API，进行真正的通话。您可以使用几行JavaScript轻松地在任何站点上安装API控制台或自行托管（需要Node.js）

RAML to HTML

RAML到HTML是一个文档工具，它根据RAML定义输出单个HTML页面控制台。它是用NodeJS编写的，可以作为命令行执行。

 
  更多信息下载

用于PHP的RAML2HTML

RAML 2 PHP for PHP是一个简单的应用程序，它使用多个模板，允许您使用RAML构建和自定义API文档。版本1包括更高级的文档功能，包括代码示例，包含Disqus评论等。

其他工具

这些只是构建API文档的一些比较流行的工具。您可以在我们的项目库中找到更多可用的工具。

RAML是唯一一个旨在以人类可读格式包含完整API生命周期的规范，其中包含代码和设计模式重用。使用RAML，您可以使用一个规范真正 设计, 构建, 测试, 记录, 和 共享 您的API。
